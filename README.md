# rubixed

A Rubik's Cube visualizer and solver, programmed in C (WIP)

## Compiling

A Makefile is already provided in this project: to compile, simple install Make and type `make` in your preferred terminal. Alternatively, use the arguments provided in the Makefile and copy the appropriate arguments for your operating system to use for the compiling command.
